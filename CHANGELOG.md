# Changelog

## 1.8.0

- 集成 vnstat 后台数据采集，页面关闭后历史数据不丢失
- 自动读取 vnstat 小时级和天级流量数据，补全图表历史
- deb 包新增 vnstat 依赖，安装时自动拉取

## 1.7.1

- 全面汉化 UI，修复英文残留

## 1.7.0

- 设置面板新增流量单位选择（自动 / B / KB / MB）
- 接口类型独立开关：Bond、VLAN、Bridge、Firewall、TAP/TUN、veth、Virtual
- 默认刷新间隔改为 1s
- WiFi 接口详情增强：SSID、频段、信道、加密方式、附近网络扫描列表
- 在线判断优化：30 秒活跃窗口，避免频繁离线闪烁
- 鼠标悬停折线图显示对应数值

## 1.5.0

- 增加折线图 tooltip 支持（主图表 + 详情弹窗图表）
- 优化在线判断逻辑

## 1.4.0

- 初始版本
- 实时流量监控与统计卡片
- 多时间跨度历史图表（Canvas 绘制）
- 接口筛选、排序、详情弹窗
- 暗色 / 明亮主题
- 10 种接口类型识别
- 迷你速率趋势 sparkline
